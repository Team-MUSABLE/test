# Height Map 계약 (H Contract) — v0.1

> **이 문서는 프로젝트의 단일 진실 공급원(single source of truth)이다.**
> A3(파이프라인)·A4(웹 핀 뷰)·A5(백엔드)·결선(FPGA)이 **전부 이 계약을 따른다.**
> 값이 흔들리면 SW↔HW 통합이 마감 직전에 깨진다. 바꿀 땐 반드시 버전을 올리고 팀에 공지.

코드 상의 실제 구현체: [`pipeline/contract.py`](../pipeline/contract.py) — 상수/직렬화 함수는 전부 여기서 import 한다. 문서와 코드가 어긋나면 **코드가 정답**, 문서를 맞춘다.

---

## 1. 격자 (Grid)

| 항목 | 값 (제안 기본값) | 비고 |
|---|---|---|
| `GRID_COLS` (x, 가로) | **48** | 손승우 최종 결정 후 확정 |
| `GRID_ROWS` (y, 세로) | **32** | 핸드헬드 디바이스 현실값 기준 |
| 총 핀 수 | 1,536 | `COLS * ROWS` |
| 순서 | **row-major** (y 바깥, x 안쪽) | `idx = y * COLS + x` |
| 원점 | 좌상단 (0,0) | 화면/촬영 정면 기준 |

> 해상도는 **결선 FPGA 실제 핀 개수와 동일**해야 한다. 웹에서 더 곱게 보이고 싶어도
> 계약 격자는 디바이스 물리 격자로 고정. 웹은 이 격자를 그대로 렌더링한다.

## 2. 값 (Height value)

- 내부 계산: `float`, **0.0 ~ 1.0** 정규화 (0 = 바닥, 1 = 최대 상승)
- 전송/저장/FPGA: **정수 0 ~ 15** (`HEIGHT_LEVELS = 16`, `HEIGHT_BITS = 4`)
- 양자화: `q = round(h_norm * (LEVELS-1))`, clamp 0..15
- **4-bit인 이유**: 결선에서 핀 1개 = 4비트로 직렬화 → 2핀/1바이트 패킹이 깔끔.

## 3. 웹/백엔드 전송 스키마 (JSON)

```json
{
  "artwork_id": "buddha_01",
  "cols": 48,
  "rows": 32,
  "levels": 16,
  "roi": { "cx": 0.5, "cy": 0.5, "scale": 1.0 },
  "data": [0, 0, 3, 7, 12, 15, 12, 7, "...길이 cols*rows, row-major, 정수 0..15"]
}
```

- `data`는 길이 `cols*rows`의 1차원 정수 배열, row-major.
- `roi`는 현재 줌/이동 상태(재현용). 핀 높이값 자체엔 영향 없음(이미 리샘플된 결과).

## 4. ROI 리샘플 (줌/이동)

- 입력: 원본 고해상도 depth/relief 맵 + ROI(중심 cx,cy, scale)
- 출력: `COLS×ROWS` 정수 H
- 규칙: **결정론적**, 다운샘플은 **max-pool**(돌출부 보존). 같은 ROI → 항상 같은 H.
- 확대 시 웹은 핀을 전부 0으로 내렸다가 새 H로 재상승 (A4 규칙).

## 5. 회화(2D) 분기

평면 회화는 depth가 없으므로 엣지/윤곽 relief로 H를 만든다. **출력 포맷은 동일**
(같은 `COLS×ROWS`, 같은 0..15). 다운스트림(웹/FPGA)은 입체/회화를 구분하지 않는다.

## 6. FPGA 직렬화 (결선 대비 — 본선 미연결, 값만 정렬)

본선에서는 연결하지 않는다. 단, **지금부터 H를 이 포맷으로 바로 뽑을 수 있게** 구조를 맞춰둔다(재작업 방지).

- 페이로드: row-major 정수 H를 4-bit 패킹 → `ceil(COLS*ROWS/2)` 바이트
- 패킹: 바이트당 [상위 니블 = 짝수 인덱스 핀, 하위 니블 = 홀수 인덱스 핀]
- 프레임(제안): `[0xA5 sync][cols][rows][levels][payload...][checksum]`
- UART 파라미터(제안): 921600 baud, 8N1 — **결선에서 손승우가 최종 확정**

구현: `contract.py::pack_4bit()` / `to_uart_frame()` (값 정렬용, 전송 코드는 결선에서).
